히어로 스킬트리 1차 - 전사의 기본

  • 워리어 마스터리

    [마스터 레벨 : 15]

    설명

    전사의 기본 소양을 길러 이동속도와 점프력, 최대 HP, 최대 이동 속도가 증가하고 적의 공격에 일정 확률로 물러서지 않는다.

  • 워리어 바디

    [마스터 레벨 : 20]

    설명

    일정 시간 동안 자신의 물리 방어력을 증가시켜주며 영구히 자신의 최대 HP를 일정 비율 증가시키고, 일정 범위 안의 적에게 피격 시 데미지가 감소한다.

  • 슬래시 블러스트

    [마스터 레벨 : 20]

    설명

    MP를 소비하여 주위의 적 다수를 동시에 공격한다.

  • 워리어 리프

    [마스터 레벨 : 10]

    설명

    점프 중 사용하면 진행 방향으로 한번 더 점프한다. 스킬레벨이 상승할수록 점프 거리가 증가한다.

  • 어퍼차지

    [마스터 레벨 : 5]

    설명

    공중으로 돌진하며 적을 공격한다. 위 방향키와 공격키를 함께 눌러서도 사용가능하다.
    커맨드 잠금 : 마우스 우클릭

  • 리프 어택

    [마스터 레벨 : 1]

    설명

    공중에서 지상으로 급강하하여 적을 습격한다. 아래 방향으로 일정 범위 안에 발판이 있어야 사용 가능하고 스킬 사용 시 좌우 방향키와 함께 사용하면 대각선으로 강하한다. 스킬 대신 아래 방향키/좌우 방향키와 공격키를 함께 눌러서도 사용 가능하다.
